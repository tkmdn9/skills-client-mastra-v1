# PDF処理スキル

## 基本手順

1. ユーザーからファイルパスを受け取る
2. `read_pdf` ツールでテキストを抽出する
3. 内容に応じて以下の形式で出力する

## 出力フォーマット

```
## ファイル概要
- ファイル: {file_path}
- ページ数: {total_pages}

## 概要
{文書全体の要約（3〜5文）}

## 主要ポイント
- {ポイント1}
- {ポイント2}
- {ポイント3}

## 文書構造
{章・セクション構成の一覧}
```

## 注意事項
- 専門用語はそのまま使用する
- 数値・日付は正確に転記する
- ページ数が多い場合は `max_pages` を指定して分割して読み込む
