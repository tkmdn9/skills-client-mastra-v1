# Excel読み取りスキル

## 基本手順

1. ユーザーからファイルパスを受け取る
2. `read_excel` ツールでファイルを読み込む
3. データの内容に応じて適切な分析・整形を行う

## データ分析の観点

読み込んだデータに対して以下を確認する：

- **構造の把握**: ヘッダー行の有無、列の意味を推定する
- **データ型**: 数値・文字列・日付の混在を確認する
- **集計が必要な場合**: 合計・平均・最大・最小などを計算する
- **複数シートの場合**: `all_sheets` に含まれる全シートを必要に応じて読み込む

## 出力フォーマット

```
## ファイル概要
- ファイル: {file_path}
- シート: {sheet_name}（全{all_sheets.length}シート）
- データ行数: {total_rows}行

## データ内容
{テーブル形式またはサマリー}

## 分析結果
{ユーザーの依頼に応じた集計・気づき}
```

## 注意事項

- `max_rows` のデフォルトは100行。大量データの場合はユーザーに確認する
- 数値の集計はツールからのデータをもとにアシスタント側で計算する
- ファイルパスはユーザーが指定したものをそのまま使用する
